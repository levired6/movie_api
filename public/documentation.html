myFlix API Documentation

Objective To build an "API" (Application Programming Interface) server-side component of a “movies” web application. The web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

Request URL Response Description Query Parameters Request Body Response Body
GET /movies 200 OK Returns a list of all movies None None JSON array of movie objects
GET /movies/:title 200 OK or 404 Not Found Returns data about a single movie by title None None JSON object of movie data
GET /users 200 OK Returns data on all the users None None JSON array of user objects
GET /users/:id 200 OK Returns data for a user by ID None None JSON object of user data
POST /users 201 Created Registers a new user None JSON object of user data JSON object of created user
PUT /users/:id 200 OK Updates user information None JSON object of updated user data JSON object of updated user data
DELETE /users/:id 200 OK Deletes a user None None Success message
POST /users/:id/favorites/:movieId 201 Created Adds a movie to user's favorites None None Success message
GET /users/:id/favorites 200 OK Returns user's favorite movies None None JSON array of movie objects
DELETE /users/:id/favorites/:movieId 200 OK Removes a movie from user's favorites None None Success message
