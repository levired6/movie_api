myFlix API Documentation

Objective To build an "API" (Application Programming Interface) server-side component of a “movies” web application. The web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

Movie API Documentation

Movie API Documentation

Request URL Response
GET / Welcome message
GET /movies List of all movies (JSON)
GET /movies/:title Data about a single movie by title (JSON)
GET /movies/genre/:genreName Data about a genre by name (JSON)
GET /movies/directorName/:directorName Data about a director by name (JSON)
POST /users Registers a new user (JSON)
PUT /users/:id Updates user info (username) (JSON)
POST /users/:id/:movieTitle Adds a movie to user's favorites (text)
DELETE /users/:id/:movieTitle Removes a movie from user's favorites (text)
DELETE /users/:id Deregisters a user (text)

Endpoint Details

Welcome Message

Returns a welcome message.

  • URL: /
  • Method: GET
  • Query Parameters: None
  • Request Body: None
  • Response: Text message. Example: 
            Welcome to myFlix app! Here are the top 10 2025 Oscar nomonated movies!

Get All Movies

Returns a list of all movies.

  • URL: /movies
  • Method: GET
  • Query Parameters: None
  • Request Body: None
  • Response: JSON array of movie objects. Example: 
            [
            {
                "title": "Anora",
                "discription": "...",
                "genre": "Comedy, Drama, Romance",
                "director": "Sean Baker",
                "imageURL": "images/Anora.jpg"
            },
            // ... more movies
        ]

Get Movie by Title

Returns data about a single movie by title.

  • URL: /movies/:title
  • Method: GET
  • Query Parameters: None
  • Request Body: None
  • Response: JSON object of the movie data. Example: 
            {
            "title": "Anora",
            "discription": "...",
            "genre": "Comedy, Drama, Romance",
            "director": "Sean Baker",
            "imageURL": "images/Anora.jpg"
        }

Get Genre by Name

Returns data about a genre by name.

  • URL: /movies/genre/:genreName
  • Method: GET
  • Query Parameters: None
  • Request Body: None
  • Response: JSON object of the genre data. All movie objects have genre objects with a Genre Description property. Example: 
            {
            "Name": "Anora",
            "Genre": "Comedy, Drama, Romance"
        }

Get Director by Name

Returns data about a director by name.

  • URL: /movies/directorName/:directorName
  • Method: GET
  • Query Parameters: None
  • Request Body: None
  • Response: JSON object of the director data. Movie objects have director objects with a directors Name. Example: 
            {
            "Name": "Sean Baker"
        }

Register New User

Registers a new user.

  • URL: /users
  • Method: POST
  • Query Parameters: None
  • Request Body: JSON object with user data. Example: 
            {
            "name": "New User"
        }
  • Response: JSON object of the newly created user (including the generated ID). Example: 
            {
            "id": "generated-uuid",
            "name": "New User"
        }

Update User Info

Updates user information (currently only username).

  • URL: /users/:id
  • Method: PUT
  • Query Parameters: None
  • Request Body: JSON object with updated user data. Example: 
            {
            "name": "Updated User Name"
        }
  • Response: JSON object of the updated user data. Example: 
            {
            "id": "user-id",
            "name": "Updated User Name"
        }

Add Movie to Favorites

Adds a movie to a user's list of favorites.

  • URL: /users/:id/:movieTitle
  • Method: POST
  • Query Parameters: None
  • Request Body: None
  • Response: Text message indicating success. Example: 
            "Movie Title has been added to user user-id's list of favorites"

Remove Movie from Favorites

Removes a movie from a user's list of favorites.

  • URL: /users/:id/:movieTitle
  • Method: DELETE
  • Query Parameters: None
  • Request Body: None
  • Response: Text message indicating success. Example: 
            "Movie Title has been removed from user user-id's list of favorites"

Deregister User

Deregisters a user.

  • URL: /users/:id
  • Method: DELETE
  • Query Parameters: None
  • Request Body: None
  • Response: Text message indicating success. Example: 
            "user user-id has been deleted"
